# hask99
Sistema de Perguntas e respostas em PHP

# Funcionalidades

 - Registo de usuários
 - Activação de conta por email
 - Recuperação da conta de usuário
 - Login de usuários
 - Publicar perguntas
 - Os usuários podem reagir e responder as perguntas
 - Notificações de comentários e reações
 - Hashtags
 - Área admnistrativa
